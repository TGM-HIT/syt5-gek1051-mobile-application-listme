# ID-11 — Favoriten

**User Story:** Als Benutzer möchte ich häufig verwendete Listen als Favoriten speichern können, damit ich sie schneller hinzufügen kann.

**Priorität:** Should Have | **SP:** 3 | **HEAD:** HIRA

---

## Overview

Users can mark lists as favourites. Favourited lists appear in a dedicated "Favoriten" section on the home screen, above the general list view, for faster access.

---

## Backend

- `favorites` table: `device_id`, `list_id` — device-scoped (no user account needed)
- `FavoriteController` — `POST /api/lists/{listId}/favorite`, `DELETE /api/lists/{listId}/favorite`, `GET /api/favorites`
- `FavoriteService` — adds/removes entries, returns favourite list IDs for the current device

---

## Frontend

- `src/services/favorite.ts` — API wrapper for add/remove/list favourites
- Home screen (`HomeView.vue`) — renders a "Favoriten" section when the device has any favourites
- Star/bookmark toggle on `ListCard.vue` or the list detail header — calls favourite service and updates local state

---

## Manual Verification

1. Open the home screen → tap the favourite icon on a list card → verify it moves to (or appears in) the "Favoriten" section.
2. Tap again to unfavourite → verify it returns to the regular section.
3. Reload the page → verify favourite state persists (backed by device-scoped API).
