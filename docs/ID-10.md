# ID-10 — Artikelbilder

**User Story:** Als Benutzer möchte ich zu einem Artikel Bilder hinzufügen, damit ich die richtigen Artikel im Geschäft finde.

**Priorität:** Should Have | **SP:** 3 | **HEAD:** PRZO

---

## Overview

Users can attach an image to any item. Images are stored in AWS S3 and the URL is persisted on the item. The frontend displays a thumbnail on the item row and allows upload from camera or gallery.

---

## Backend

- `Item` entity: `imageUrl` (TEXT, nullable)
- `ImageController` — `POST /api/images/upload` accepts multipart file, stores to S3, returns the public URL
- `ItemController` — `imageUrl` accepted in `CreateItemRequest` / `UpdateItemRequest`
- S3 bucket configured via `application.properties` (`aws.s3.bucket`, `aws.region`)

---

## Frontend

- `src/services/image.ts` — wraps `POST /api/images/upload`, returns URL
- `src/components/item/ImagePicker.vue` — camera/gallery picker, calls image service, passes URL up
- `ItemRow.vue` — renders a small thumbnail when `item.imageUrl` is set
- Add/edit item sheet — includes the `ImagePicker` component

---

## Manual Verification

1. Open a list → add/edit an item → tap the image picker → select a photo.
2. Verify the image uploads and a thumbnail appears on the item row.
3. Reload → verify the image persists (URL stored on item).
4. Remove the image → verify the thumbnail disappears.
