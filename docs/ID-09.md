# ID-09 — Mengen/Einheiten und Labels

**User Story:** Als Benutzer möchte ich zu Artikeln Mengen/Einheiten (z.B. „2 kg", „500 ml") und Labels (z.B. „Bio", „Vegetarisch") hinzufügen können.

**Priorität:** Should Have | **SP:** 3 | **HEAD:** PETP

---

## Overview

Items can carry an optional quantity with a unit (e.g. `2 kg`, `500 ml`) and one or more colour-coded labels. Both are stored on the backend and synced in real time.

---

## Backend

- `Item` entity: `quantity` (BigDecimal, nullable), `quantityUnit` (String, nullable)
- `Label` entity: `id`, `name`, `color` (hex), `listId` — scoped per list
- `LabelController` — CRUD for labels (`GET/POST/DELETE /api/lists/{listId}/labels`)
- `ItemController` — `quantity` and `quantityUnit` accepted in `CreateItemRequest` / `UpdateItemRequest`
- `ItemLabelController` — assign/remove labels on items (`PUT/DELETE /api/lists/{listId}/items/{itemId}/labels/{labelId}`)

---

## Frontend

- `src/services/label.ts` — API wrapper for label CRUD
- `src/stores/labels.ts` — Pinia store; fetched when list detail opens
- `src/components/item/LabelTag.vue` — coloured badge rendered per label on `ItemRow`
- `src/components/item/LabelPicker.vue` — dropdown for assigning labels when editing an item
- Quantity/unit fields rendered in the add/edit item sheet

---

## Manual Verification

1. Open a list → add an item with quantity `2` and unit `kg` → verify it shows on the item row.
2. Create a label "Bio" with a green colour → assign it to an item → verify badge appears.
3. Remove a label assignment → verify badge disappears.
4. Reload the page → verify quantity and labels persist.
