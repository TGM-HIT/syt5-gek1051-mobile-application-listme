# ID-15 — Dark Mode / helles Design

**User Story:** Als Benutzer möchte ich Dark Mode / helles Design umschalten können, damit die App für mich angenehmer zu bedienen ist.

**Priorität:** Should Have | **SP:** 3 | **HEAD:** SIEL

---

## Overview

The app supports a dark and a light theme. The preference is persisted in `localStorage` and applied immediately on load to avoid a flash of the wrong theme.

---

## Implementation

- **Theme system:** Tailwind CSS 4 with Catppuccin Frappe (dark) and Catppuccin Latte (light) palettes via CSS custom properties on `:root`
- `src/stores/theme.ts` — Pinia store: `isDark` ref, `toggle()`, persists to `localStorage` under `listme:theme`
- `src/views/SettingsView.vue` — toggle switch for dark/light mode
- `index.html` / `main.ts` — reads the stored preference and sets the `dark` class on `<html>` before Vue mounts (no flash)

---

## Manual Verification

1. Open Settings → toggle to light mode → verify the whole UI switches colour palette.
2. Reload → verify the chosen theme is still active.
3. Toggle back to dark mode → verify it switches immediately.
