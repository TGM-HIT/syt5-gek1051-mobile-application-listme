# ID-16 — Export (CSV / PDF)

**User Story:** Als Benutzer möchte ich Einkaufslisten exportieren können (CSV/PDF), damit ich meine Daten sichern oder extern auswerten kann.

**Priorität:** Should Have | **SP:** 3 | **HEAD:** PRZO

---

## Overview

Users can export any list as a CSV file (machine-readable) or a PDF (print-ready). Both formats include item name, quantity, unit, price, category, and checked state.

---

## Backend

- `ExportController`
  - `GET /api/lists/{listId}/export/csv` — streams a `text/csv` response
  - `GET /api/lists/{listId}/export/pdf` — streams an `application/pdf` response using OpenPDF (`com.github.librepdf:openpdf`)
- `ExportService` — builds the CSV string and PDF document from the list's items

---

## Frontend

- `src/services/export.ts` — calls the export endpoints, triggers a browser download via a temporary `<a>` element with `download` attribute
- List detail header menu → "Als CSV exportieren" / "Als PDF exportieren"

---

## Manual Verification

1. Open a list with several items → tap "Als CSV exportieren" → verify a `.csv` file downloads with the correct columns.
2. Tap "Als PDF exportieren" → verify a `.pdf` file downloads and is readable.
3. Export an empty list → verify the file contains only the header row (CSV) or a blank table (PDF).
