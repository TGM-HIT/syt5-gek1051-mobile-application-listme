# ID-14 — Preise und Budgetübersicht

**User Story:** Als Benutzer möchte ich Preise zu Artikeln speichern und die geschätzten Gesamtkosten pro Liste sehen, damit ich mein Budget im Blick behalte.

**Priorität:** Should Have | **SP:** 3 | **HEAD:** SIEL

---

## Overview

Items can have an optional price. The list detail view shows a budget bar with the total estimated cost, broken down by category. The total updates in real time as items are checked off or prices change.

---

## Backend

- `Item` entity: `price` (BigDecimal, nullable)
- `ItemController` — `price` accepted in `CreateItemRequest` / `UpdateItemRequest`
- `BudgetController` — `GET /api/lists/{listId}/budget` returns total, by-category breakdown, and checked-only subtotal

---

## Frontend

- `src/services/budget.ts` — wraps `GET /api/lists/{listId}/budget`
- `src/components/list/BudgetBar.vue` — horizontal bar showing spent vs. total; turns red when over a soft limit
- List detail view renders `BudgetBar` below the item list when at least one item has a price

---

## Manual Verification

1. Add items with prices to a list → verify the budget bar appears with the correct total.
2. Check off an item → verify the "spent" portion of the bar updates.
3. Add a price via edit → verify the total updates.
4. Set all prices to null → verify the budget bar disappears.
